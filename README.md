# さくらさくら カラオケWebアプリ

自作した `sakurasakura01.mp4` を再生し、動画の下に大きく歌詞を固定表示するシンプルなWebアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 動画を再生します。
3. 下に表示される大きな歌詞を見ながら歌唱します。
4. 「最初から」で動画を最初から再生し、「やりなおす」で動画を先頭に戻して停止します。

## 動画

アプリ内の動画ファイル:

- `assets/sakurasakura01.mp4`

## 歌詞

- のやまも　さとも
- みわたす　かぎり
- かすみか　くもか
- あさひに　におう
- さくら　さくら
- はなざかり
